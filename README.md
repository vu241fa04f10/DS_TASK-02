# PRODIGY_DS_02

## Task-02: Data Cleaning and Exploratory Data Analysis (EDA)

This project is part of the Data Science Internship tasks at Prodigy InfoTech.

## Objective

Perform data cleaning and exploratory data analysis (EDA) on a dataset of choice to identify patterns, relationships, and trends in the data.

## Dataset Used

Titanic Dataset

Dataset Link:  
https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%202

## Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Steps Performed

1. Imported required libraries  
2. Loaded Titanic dataset using pandas  
3. Checked dataset structure and missing values  
4. Filled missing values in Age and Embarked columns  
5. Removed Cabin column due to excessive missing values  
6. Removed duplicate rows  
7. Performed exploratory data analysis using graphs

## Exploratory Analysis Performed

- Survival count analysis
- Gender vs Survival comparison
- Passenger Class vs Survival
- Age distribution
- Fare distribution
- Correlation heatmap

## Key Insights

- Female passengers had higher survival rates.
- First class passengers survived more than lower classes.
- Younger passengers had better survival chances.
- Higher fare passengers showed better survival trends.
